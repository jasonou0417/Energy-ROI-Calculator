# ⚡ 企業電費試算與儲能財務決策系統 (V18)

這是一個專為企業用電大戶設計的財務評估工具。只要上傳台電的 15 分鐘用電明細，系統就能自動比對「現況 (三段式)」與「未來 (批次生產+儲能)」的電費差異，並精算完整的 15 年專案 IRR (內部報酬率)。

---

## 🚀 快速上手教學

### 第一步：上傳用電資料
1. 進入系統網頁後，找到中間的 **「請上傳 15 分鐘用電明細 (CSV)」** 區塊。
2. 將台電高壓入口網下載的「15 分鐘用電量 CSV 檔」拖曳進去。
3. 系統支援多種編碼，上傳後會自動開始解析一整年的數據。

### 第二步：設定左側合約與費率參數
系統左側面板包含了所有的模擬變數，請依照公司的實際狀況進行調整：

* **⚙️ 1. 契約容量設定：** * 展開「現況」與「未來」區塊，分別填入不建置儲能與建置儲能後的合約容量（如：經常、半尖峰等）。
* **💰 2. 台電費率設定：**
  * 預設為最新版台電費率。若未來遇到台電漲價，可直接在此修改尖峰/離峰單價，右側的財務報表會 **即時重新計算**。
* **🔋 3. 儲能櫃與財務參數：**
  * **容量與功率：** 設定儲能設備規格（預設 2349 kWh / 1125 kW）。
  * **放電模式：** 建議選擇「定額放電」並設定功率（如 316kW）以穩定削峰。
  * **財務參數：** 輸入廠商報價（CAPEX）、預估維護費（OPEX）與電池衰退率，這是計算真實 IRR 的核心。

*(💡 提示：所有參數修改後，右側的數據面板都會在 2~3 秒內自動更新，不需重新上傳檔案！)*

## 📊 如何閱讀財務報表？

分析完成後，網頁右側會產生三個決策區塊：

1. **🏛️ 專案總覽與財務指標：**
   * 直接顯示「總淨利 (NPV)」、「投資報酬率 (ROI)」，以及「第一年總節省金額」。
2. **📋 節省金額明細：**
   * 以「項目差額法」顯示：基本費省多少？流動費省多少？超約費省多少？這能幫助釐清儲能櫃的價值來源。
3. **💰 0~N 年專案現金流與回收狀況：**
   * 扣除每年的維護費 (OPEX) 與電池衰退後，算出**「真實動態回收期」**（例如 6.8 年）。
   * 附帶完整的逐年現金流表格，清楚看見何時轉虧為盈。

## 📥 匯出與報告
在網頁的最下方，您可以點擊下載按鈕：
* **📊 下載【每月節省金額明細】CSV**
* **💰 下載【0~N年完整現金流】CSV**

下載後的檔案為 `utf-8-sig` 編碼，直接用 Excel 開啟不會亂碼，方便您貼入簡報向高層報告。
